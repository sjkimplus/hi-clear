## 프로젝트 기획 및 목적

-----------------------
- 배드민턴 동호인들이 **정기적 모임**과 **일회성 모임**으로 배드민턴을 칠 수 있도록 관리해주는 서비스
- **위치기반 체육관, 번개, 클럽 조회기능** 제공
- 유저들간 평가를 통해 **실력 및 매너점수 평가** 기능 제공
<br><br>


## 핵심 기능

-----------------------
<h3>SSE 알림 기능</h3>

<h3>CI/CD 파이프라인 구축</h3>
